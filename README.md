#Global Air Quality Analysis

##Project Overview

This project analyzes air quality data from major cities around the world using Python. The main focus is on two important pollutants:

* **PM2.5** – fine particles that can affect human health.
* **NO₂** – a gas mainly produced by vehicles and industries.

The goal of this project is to understand pollution trends, compare cities, and identify patterns using data analysis and visualization.

##Dataset Information

The dataset contains:

* City and country names
* Latitude and longitude
* Year and month of measurement
* PM2.5 and NO₂ values
* Data quality and source information

##Tools and Libraries Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

##Analysis Performed

* Data loading and inspection
* Missing value and duplicate checks
* PM2.5 distribution analysis
* NO₂ distribution analysis
* Yearly and monthly pollution trends
* City-wise pollution comparison
* Top 10 most polluted cities
* Top 10 cleanest cities


##How to Run

1. Open the notebook in Google Colab.
2. Upload the `air_quality_global.csv` file.
3. Run all cells to generate the analysis and visualizations.
